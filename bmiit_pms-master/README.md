**BMIIT Project Management System**

 Overview
Web-based system to manage student projects, guides, milestones, and submissions.

## Setup

1. Clone the repo:
git clone https://github.com/Neel-Salot/bmiit_pms.git
2. Set up the database:
- Open phpMyAdmin
- Create a database `bmiit_pms`
- Run SQL scripts in `/db` folder in order
3. Configure database connection in `includes/db.php`

## Usage
- Open `login.php` in browser
- Use seeded users (or create your own via SQL scripts)

## Version Control
- Commit changes for new features or bug fixes
- Push to GitHub regularly
- Branching recommended for new features
